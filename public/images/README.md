# Imagens do Projeto ORMED

## Logo
- O logo SVG foi criado baseado na identidade visual da ORMED
- Cores: Azul marinho (#002147) e Dourado (#FFD700)
- Para usar o logo oficial, substitua o arquivo `logo.svg`

## Fotos da Bastonária
- A foto oficial está disponível em:
  https://www.ordemdosmedicos.ao/assets/imagens/about/aboutimg02.jpg
- Para usar localmente, baixe e salve como `bastonaria.jpg` nesta pasta
- Ou mantenha a URL externa no arquivo `lib/data.ts`

## Outras Imagens
- Hero images: Crie imagens de 1920x1080 para o banner principal
- News images: Crie imagens de 800x450 para as notícias
- Event images: Crie imagens de 800x450 para os eventos
- Avatar images: Crie imagens de 200x200 para os testemunhos

## Formato Recomendado
- Use imagens WebP para melhor performance
- Ou JPG/PNG com compressão otimizada
